# Instructions
Prerequisites:
- Hugo Extended (recommended: `v0.83.1` for this project)
- Go (required for Hugo modules)

Preview website locally:
```shellscript
hugo server -D
```

Production build:
```shellscript
hugo --gc --minify --cleanDestinationDir
```

If deploying to a specific domain, set base URL during build:
```shellscript
hugo --gc --minify --cleanDestinationDir -b "https://<your-domain>/"
```

Then, publish with:
```shellscript
cd public
git add .
git commit -m "website updated"
git push https://github.com/kieranjwood/kieranjwood.github.io.git
```
